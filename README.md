# 🧮 Interactive Calculator

A simple command-line calculator built in **Python** — my first project for the **Code to AI** course! 🚀

It asks the user for two numbers and instantly shows the results of all the basic math operations.

---

## ✨ What it does

- Takes **two numbers** from the user
- Calculates and prints:
  - ➕ Addition
  - ➖ Subtraction
  - ✖️ Multiplication
  - ➗ Division (rounded to 2 decimals)
  - 🔢 Power (`**`)
  - 📐 Remainder (`%`)

---

## 🧠 Concepts I used

- `input()` to take user input
- **Type casting** (`float()`) to turn text into numbers
- **Variables** to store values
- **Arithmetic operators** (`+ - * / ** %`)
- **f-strings** for clean, formatted output

---

## ▶️ How to run

1. Make sure **Python** is installed.
2. Download or clone this repo.
3. Open a terminal in the project folder and run:

```bash
python calculator.py
```

4. Type two numbers when asked — and see the magic! ✨

---

## 📸 Example

```
===================================
   🧮  WELCOME TO MY CALCULATOR  🧮
===================================
Enter the first number : 8
Enter the second number: 2

--------- RESULTS ---------
8.0 + 2.0  = 10.0

8.0 * 2.0  = 16.0
8.0 / 2.0  = 4.00

```
